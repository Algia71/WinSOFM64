# WinSOFM64

Applicazione della quantizzazione vettoriale operata dalla Self Organizing Features Map (SOFM) di Kohonen sulla distribuzione omogenea di punti nel piano.

_Application of vector quantization performed by Kohonen's Self Organizing Features Map (SOFM) on the homogeneous distribution of points in the plane._
